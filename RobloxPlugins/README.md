# Roblox Plugins

Here's a folder of some plugins I have developed for Roblox development. The packages these rely on have been removed as I don't own them.

- LuauExplorer.lua; My version of an Explorer window similar to the new Luau based one developed by Roblox. This can be quite laggy so be careful. The search bar is also quite buggy. This will be updated with time.
